## Installing dependencies
libzlib and libpng

#### Linux dependencies

##### debian or ubuntu based distros
```sh
sudo apt install libzlib libpng
```

##### arch based distros
```sh
sudo pacman -syu libzlib libpng
```

##### fedora or rhel or centos
```sh
sudo rpm install libzlib libpng
```
